# 使用说明书

## 文档定位

- machine_readable_tag: aicopilotcb-usage-manual-v1
- 文档角色：AICopilotCB 日常使用、分流、维护与校验的统一说明书

## 快速起手

1. 新开会话先走 `.github/prompts/workspace-chat-start.prompt.md`。
2. 若目标是维护控制面本身，走 `.github/prompts/maintain-collaboration-scheme.prompt.md`。
3. 若目标是 readiness 或汇报，读取 `docs/AUTO_DEV_NAVIGATION.md` 后进入对应模板或清单。

## 基本原则

- 入口先分流，再执行，不混用多个主入口。
- 默认以 `workspace-init.json`、`role-boundaries.json`、`active-context.json` 为配置真源。
- 受保护 Markdown 写入后必须执行 guard 校验。

## 常用入口

- 需求索引：`docs/AI协同方案需求大纲索引.md`
- 运行导航：`docs/AUTO_DEV_NAVIGATION.md`
- 初始化配置说明：`docs/初始化配置与运行时上下文说明.md`
- 全局规则：`.github/copilot-instructions.md`
